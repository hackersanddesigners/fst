(Personal) Information about the author


  • Relator term

  • Titles held by the author

  • Authors dead/alive


Profession of the author

This filter indicates in what relation the person listed here stands toward the publication.* An example would for instance be whether the person is an author, editor or interviewee.


*This filter is an interpretation of tag 100 - Main Entry-Personal Name (NR), subfield code $e - Relator term (R) .


Titles held by the author

This filter indicates which titles and/or words are associated with the name of the author.* An example here would for instance be the profession of an author, such as journalist or linguist.


*This filter is an interpretation of tag 100 - Main Entry-Personal Name (NR), subfield code $c - Titles and words associated with a name (R).


Authors dead/alive

This filter indicates a certain time period associated with the name of an author.* It is usually filled in if two names resemble each other to distinguish between them. The time periods registered here could either indicate the lifespan of an author, and as such whether the author is dead or alive, but could also indicate a period of flourishing. This is a potential limitation of our interpretation of this field.


*This filter is an interpretation of tag 100 - Main Entry-Personal Name (NR), subfield code $d - Dates associated with a name (NR).
