This filter refers to the producing entity of a publication.* It might be used interchangeably with the publisher of a publication, because not all MARC 21 tags make a clear distinction between the publisher and the manufacturer of an item.


*This field is an interpretation of tag 260 - Publication, Distribution, etc. (Imprint) (R), subfield code $f - Manufacturer (R) and/or tag 264 - Production, Publication, Distribution, Manufacture, and Copyright Notice (R), subfield code $b - Name of producer, publisher, distributor, manufacturer (R).
