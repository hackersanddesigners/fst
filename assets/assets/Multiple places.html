This filter refers to those works that have been published in more than one location.*


*This filter is based on 260 - Publication, Distribution, etc. (Imprint) (R), subfield code $a - Place of publication, distribution, etc. (R), as well as tag 264 - Production, Publication, Distribution, Manufacture, and Copyright Notice (R), subfield code $a - Place of production, publication, distribution, manufacture (R), and considers those items for which there is more than one place of publishing provided.
